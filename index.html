Matrix Zeroes

In JavaScript, "set matrix zeros" refers to the task of modifying a two-dimensional array (matrix) in such a way that if any element in the matrix is zero, its entire row and column are set to zero.

This operation involves iterating through the matrix to identify zero elements, and then updating the corresponding rows and columns accordingly. The goal is to achieve this transformation efficiently, preferably in a single pass through the matrix, without using additional memory proportional to the size of the matrix.
